# 📝 To-Do List App

A simple, clean, and responsive **To-Do List web app** built using **HTML, CSS, and JavaScript**.
Designed for simplicity, speed, and a smooth user experience.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit tasks inline
* ✅ Mark tasks as complete (strike-through)
* 🗑️ Remove tasks
* ⌨️ Add tasks using the **Enter key**
* 📱 Fully responsive design
* 🎨 Minimal and playful UI (custom font + soft colors)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla)** – Functionality

## ⚙️ How It Works

### Adding a Task

* Type into the input field
* Press **Enter** or click **Add**
* Task gets added to the list

### Editing a Task

* Click the ✏️ edit button
* Modify the text directly
* Click again or click outside to save

### Completing a Task

* Click the checkbox
* Task gets a **line-through effect**

### Deleting a Task

* Click the 🗑️ button
* Task is removed instantly
